## Описание

В ходе выполнения данной лабораторной работы были углубленно изучены основные управляющие конструкции и структуры данных в JavaScript. Рассмотрены различные виды циклов (`for`, `while`, `do...while`), операторы управления циклом (`break`, `continue`) и вложенные циклы. Особое внимание уделено функциям: классическое объявление, функции без возвращаемого значения, параметры по умолчанию, функции как значения, замыкания (closure) и современный синтаксис стрелочных функций (arrow functions). Были изучены массивы: способы создания, добавление и удаление элементов (`push`, `pop`, `shift`, `unshift`), перебор с помощью цикла `for...of`, поиск элементов (`indexOf`, `includes`), а также возможность хранения в массиве данных разных типов. В финальной части работы рассмотрены объекты: создание через литерал объекта, доступ к свойствам, добавление и удаление свойств "на лету", создание методов объекта, перебор свойств с помощью `for...in`, а также работа с вложенными структурами данных (объекты и массивы внутри объектов).
